# The Shadow of Sword ⚔️

**The Shadow of Sword** is an action-adventure game project developed using the Unity engine. Originally created as a project at the University of Science (VNUHCM), this repository contains the core game logic, assets, and custom shaders that bring the game's world to life.

## 🛠 Tech Stack & Tools

* **Game Engine:** Unity *(Check `ProjectSettings/ProjectVersion.txt` for the specific editor version)*
* **Programming Language:** C# (67.7%)
* **Graphics & Shaders:** ShaderLab (27.5%) & HLSL (4.8%) for custom visual effects, dynamic lighting, and stylized material rendering.

## ✨ Features
*(Note: You can update these based on the final scope of your game)*
* **Combat Mechanics:** Responsive sword-fighting controls and character interactions.
* **Custom Visuals:** Tailored atmospheric environments utilizing custom ShaderLab scripts.
* **Modular Architecture:** Organized C# scripts allowing for easy expansion of game systems.

## 🚀 Getting Started

### Prerequisites
* [Unity Hub](https://unity.com/download)
* A compatible IDE (Visual Studio, VS Code, or Rider) for C# development.

### Installation & Setup
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/tblong0210/the-shadow-of-sword-hcmus.git
    ```
2.  **Open the project in Unity:**
    * Launch **Unity Hub**.
    * Click on **Open** -> **Add project from disk**.
    * Navigate to the cloned `the-shadow-of-sword-hcmus` directory and select it.
    * Allow Unity to resolve packages and import assets (this may take a few minutes on the first run).
3.  **Load the Game:**
    * Once the Unity Editor opens, navigate to the `Assets/` folder in the Project window.
    * Locate your main entry Scene and double-click to open it.
    * Press the **Play** button at the top of the editor to run the game.

## 🎮 Controls (Default)
* **W, A, S, D:** Movement
* **Left Mouse Click:** Attack / Interact
* **Spacebar:** Jump / Dodge
* **Esc:** Pause Game

## 📂 Repository Structure
* `Assets/`: Contains all game resources including C# Scripts, Prefabs, Scenes, Materials, and Models.
* `Packages/`: Unity package dependencies required for the project.
* `ProjectSettings/`: Unity configuration files (Input, Physics, Graphics, Tags/Layers, etc.).
